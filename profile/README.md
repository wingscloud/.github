### 设计理念

面向前端工程师独立开发应用程序的场景，提供一套轻量级、可灵活扩展的 Web 及服务端应用程序解决方案。Wings Cloud 在源码上全部采用 TypeScript / JavaScript 及成熟的生态环境，减少开发人员学习成本及心智负担。
